# 缘遇 — 产品设计文档

> 本文档记录缘遇 APP 的完整产品设计，包括产品定位、功能架构、用户流程、UI 规范等。

---

## 1. 产品定位

### 1.1 产品简介

**缘遇**是一款面向 18-30 岁都市年轻人的移动社交应用，主打"兴趣驱动 + 真实社交"。

### 1.2 核心价值主张

- **真实高效**：通过兴趣匹配和地理位置筛选，快速找到志同道合的人
- **温暖连接**：不只是"左滑右滑"，通过兴趣社群建立更深的关系
- **安全可信赖**：实名认证体系 + 社区自治，保障真实社交体验

### 1.3 目标用户画像

- **小雅**，23 岁，深圳互联网运营
  - 喜欢跑步和咖啡，周末活跃
  - 希望拓宽社交圈，认识有趣的人
  - 对社恐友好，偏好线上先聊再约

- **阿杰**，27 岁，广州产品经理
  - 喜欢后摇和户外徒步
  - 工作忙，碎片时间刷一刷
  - 希望遇到有共同兴趣的另一半

## 3. 核心页面设计

### 3.1 登录/注册页

**设计原则：** 极简无障碍，降低首次使用门槛。

- 默认"一键注册"模式（首次使用）
- 切换"已有账号登录"（老用户）
- 昵称输入 → 2-12 字实时校验
- 按钮文案随模式动态变化
- 注册即表示同意用户协议和隐私政策

**视觉风格：** 渐变背景（#FFF5F5 → #FAFAFA），圆角卡片居中，品牌珊瑚红主色。

### 3.2 发现页（滑动卡片）

**设计原则：** Tinder 式滑动交互，降低决策成本。

- 卡片堆叠展示（当前卡片 + 下一张预览）
- 左滑 = 跳过 / 右滑 = 喜欢 / 上滑 = 超级喜欢
- 卡片显示：头像、昵称、年龄、城市、个人简介
- 底部固定操作栏：跳过 / 喜欢 / 超级喜欢
- 匹配成功后居中弹出模态框，点击进入聊天

### 3.3 聊天页

**设计原则：** 简洁对话体验，突出消息内容。

- 顶部固定标题栏 + 返回按钮
- 消息气泡：自己居右（品牌色背景），对方居左（白色背景）
- 底部输入栏 + 发送按钮
- 支持 Enter 键发送
- 新消息自动滚动到底部

### 3.4 消息列表页

**设计原则：** 高效定位会话，快速发起聊天。

- 按最后消息时间倒序
- 每行显示：头像 + 昵称 + 最后消息 + 时间 + 未读数
- 点击进入聊天页
- 空状态引导去发现页匹配

### 3.5 社群页

**设计原则：** 兴趣导向的社群发现和浏览。

- Tab 切换：全部 / 运动 / 音乐 / 户外 / 美食
- 卡片展示：封面图 + 社群名 + 简介 + 成员数
- 点击进入社群详情：成员列表 + 活动列表 + 加入按钮

### 3.6 个人资料页

**设计原则：** 展示真实的自己，管理个人形象。

- 顶部：头像 + 昵称 + 在线状态
- 信息区：性别、年龄、城市、个人简介
- 兴趣标签（可编辑）
- 真心话问答展示
- 编辑资料入口

## 4. UI 设计规范

### 4.1 色彩系统

| 变量 | 色值 | 用途 |
|------|------|------|
| `--primary` | #FF6B6B | 品牌色、主按钮、重点强调 |
| `--secondary` | #4ECDC4 | 辅助色、超级喜欢、在线标识 |
| `--background` | #FAFAFA | 页面背景 |
| `--surface` | #FFFFFF | 卡片、浮层背景 |
| `--text` | #333333 | 主文字 |
| `--text-sub` | #888888 | 次要文字、时间戳 |
| `--text-light` | #AAAAAA | 占位符、弱提示 |
| `--error` | #FF4757 | 错误提示 |
| `--success` | #2ED573 | 成功提示 |
| `--border` | #EEEEEE | 分割线、边框 |

### 4.2 字体规范

- 主字体：系统默认字体栈 `-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Helvetica Neue', sans-serif`
- 标题：16-32px，font-weight: 700
- 正文：14-16px，font-weight: 400
- 说明：11-13px，color: text-sub

### 4.3 间距规范

- 页面水平内边距：16px
- 卡片内边距：16px
- 元素间距基准：8px（8/16/24/32/40/48）
- 圆角：按钮 26px（大）/ 12px（小），卡片 16px，输入框 12px

### 4.4 阴影

```css
--shadow: 0 2px 8px rgba(0,0,0,0.08);
--shadow-lg: 0 8px 24px rgba(255,107,107,0.3);
```

## 5. 数据指标

### 5.1 核心指标

| 指标 | 目标 |
|------|------|
| 日活跃用户（DAU） | 目标 10,000+ |
| 用户注册转化率 | > 60% |
| 匹配率（滑动→匹配） | > 15% |
| 消息发送率（匹配→发消息） | > 70% |
| 社群加入率 | > 40% |
| 次日留存率 | > 45% |

### 5.2 冷启动策略

- 种子用户：邀请 50 名真实用户入驻
- 社群建设：预设 10 个高质量兴趣社群
- 内容填充：运营团队模拟部分对话和活动
- 推荐优化：基于兴趣和地理位置的双重推荐

---

## 6. 隐私与安全

- 用户可自主设置个人资料的可见范围
- 支持拉黑和举报机制
- 聊天内容采用 HTTPS 加密传输
- JWT Token 短期有效，自动刷新
- 敏感操作需二次验证
